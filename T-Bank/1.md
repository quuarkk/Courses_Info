### 2. **–°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∏**
#### **–ü—É–∑—ã—Ä—å–∫–æ–≤–∞—è —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ (Bubble Sort)**
**–ö–∞–∫ —Ä–∞–±–æ—Ç–∞–µ—Ç?**  
–°—Ä–∞–≤–Ω–∏–≤–∞–µ—Ç —Å–æ—Å–µ–¥–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç—ã –∏ –º–µ–Ω—è–µ—Ç –∏—Ö –º–µ—Å—Ç–∞–º–∏, –µ—Å–ª–∏ –æ–Ω–∏ –≤ –Ω–µ–ø—Ä–∞–≤–∏–ª—å–Ω–æ–º –ø–æ—Ä—è–¥–∫–µ.  
**–°–ª–æ–∂–Ω–æ—Å—Ç—å:** O(n¬≤).  
**–ê–Ω–∞–ª–æ–≥–∏—è:** –ö–∞–∫ –ø—É–∑—ã—Ä—å–∫–∏ –≤ –≤–æ–¥–µ –ø–æ–¥–Ω–∏–º–∞—é—Ç—Å—è –Ω–∞–≤–µ—Ä—Ö.  
**–ö–æ–¥:**
```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

print(bubble_sort([64, 34, 25, 12, 22, 11, 90]))  # [11, 12, 22, 25, 34, 64, 90]
```

#### **–°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ —Å–ª–∏—è–Ω–∏–µ–º (Merge Sort)**
**–ö–∞–∫ —Ä–∞–±–æ—Ç–∞–µ—Ç?**  
–î–µ–ª–∏—Ç –º–∞—Å—Å–∏–≤ –Ω–∞ –¥–≤–µ —á–∞—Å—Ç–∏, —Å–æ—Ä—Ç–∏—Ä—É–µ—Ç –∫–∞–∂–¥—É—é, –∑–∞—Ç–µ–º –æ–±—ä–µ–¥–∏–Ω—è–µ—Ç –∏—Ö.  
**–°–ª–æ–∂–Ω–æ—Å—Ç—å:** O(n log n).  
**–ê–Ω–∞–ª–æ–≥–∏—è:** –°–æ–±–∏—Ä–∞–µ—Ç–µ –ø–∞–∑–ª ‚Äî —Å–Ω–∞—á–∞–ª–∞ —Å–æ—Ä—Ç–∏—Ä—É–µ—Ç–µ –¥–µ—Ç–∞–ª–∏ –ø–æ —Ü–≤–µ—Ç–∞–º, –∑–∞—Ç–µ–º –æ–±—ä–µ–¥–∏–Ω—è–µ—Ç–µ.  
**–ö–æ–¥:**
```python
def merge_sort(arr):
    if len(arr) > 1:
        mid = len(arr) // 2
        left = arr[:mid]
        right = arr[mid:]

        merge_sort(left)
        merge_sort(right)

        i = j = k = 0
        while i < len(left) and j < len(right):
            if left[i] < right[j]:
                arr[k] = left[i]
                i += 1
            else:
                arr[k] = right[j]
                j += 1
            k += 1

        while i < len(left):
            arr[k] = left[i]
            i += 1
            k += 1

        while j < len(right):
            arr[k] = right[j]
            j += 1
            k += 1
    return arr

print(merge_sort([38, 27, 43, 3, 9, 82, 10]))  # [3, 9, 10, 27, 38, 43, 82]
```

#### **–°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –ø–æ–¥—Å—á–µ—Ç–æ–º (Counting Sort)**
**–ö–∞–∫ —Ä–∞–±–æ—Ç–∞–µ—Ç?**  
–ü–æ–¥—Å—á–∏—Ç—ã–≤–∞–µ—Ç –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –≤—Ö–æ–∂–¥–µ–Ω–∏–π –∫–∞–∂–¥–æ–≥–æ —ç–ª–µ–º–µ–Ω—Ç–∞, –∑–∞—Ç–µ–º –≤–æ—Å—Å—Ç–∞–Ω–∞–≤–ª–∏–≤–∞–µ—Ç –º–∞—Å—Å–∏–≤.  
**–°–ª–æ–∂–Ω–æ—Å—Ç—å:** O(n + k), –≥–¥–µ k ‚Äî —Ä–∞–∑–º–µ—Ä –∞–ª—Ñ–∞–≤–∏—Ç–∞ (–¥–∏–∞–ø–∞–∑–æ–Ω –∑–Ω–∞—á–µ–Ω–∏–π).  
**–ê–Ω–∞–ª–æ–≥–∏—è:** –°—á–∏—Ç–∞–µ—Ç–µ, —Å–∫–æ–ª—å–∫–æ —è–±–ª–æ–∫, –≥—Ä—É—à –∏ –±–∞–Ω–∞–Ω–æ–≤ –≤ –∫–æ—Ä–∑–∏–Ω–µ, –∑–∞—Ç–µ–º —Ä–∞—Å–∫–ª–∞–¥—ã–≤–∞–µ—Ç–µ –∏—Ö –ø–æ –ø–æ—Ä—è–¥–∫—É.  
**–ö–æ–¥:**
```python
def counting_sort(arr):
    max_val = max(arr)
    count = [0] * (max_val + 1)
    output = []

    for num in arr:
        count[num] += 1

    for i in range(len(count)):
        output.extend([i] * count[i])

    return output

print(counting_sort([4, 2, 2, 8, 3, 3, 1]))  # [1, 2, 2, 3, 3, 4, 8]
```

### 3. **–ë–∏–Ω–∞—Ä–Ω—ã–π –ø–æ–∏—Å–∫**
**–ö–∞–∫ —Ä–∞–±–æ—Ç–∞–µ—Ç?**  
–ò—â–µ—Ç —ç–ª–µ–º–µ–Ω—Ç –≤ –æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–º –º–∞—Å—Å–∏–≤–µ, –¥–µ–ª—è –µ–≥–æ –ø–æ–ø–æ–ª–∞–º –Ω–∞ –∫–∞–∂–¥–æ–º —à–∞–≥–µ.

```python
def binary_search(arr, target):
    left = 0
    right = len(arr) - 1

    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1

arr = [1, 3, 5, 7, 9, 11]
print(binary_search(arr, 9))  # –ò–Ω–¥–µ–∫—Å 4
```

    right = 10**18  # –ë–æ–ª—å—à–æ–µ —á–∏—Å–ª–æ –¥–ª—è –≤–µ—Ä—Ö–Ω–µ–π –≥—Ä–∞–Ω–∏—Ü—ã
